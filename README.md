### Actividad:
> Realizar un sitio web que contenga las propiedades esenciales del CSS. No es necesario el diseño responsive.
> Debe tener un diseño agradable y eficaz siguiendo los principios de UI/UX. Tiene que tener como mínimo 4 secciones o páginas, siendo una de ellas el inicio o landing page, y otra donde estará ubicado el formulario. Se debe utilizar una base de datos. Aplicar efectos con css. Recomiendo utilizar al máximo las propiedades css para mejorar la visual de la pagina.
